## DSK System

---

    這是 Donseking 寫的命令操作系統  
    要使用命令需要前綴 "-"  

    直接打 "math" 會切換成 math 模式  
    math 模式前綴 "#"   
                       2022.06.30

---

### dsk.py

命令 : 

pit   > 輸出文字  

    -pit Hello

et    > 關閉程式

    -et
    
path  > 查看當前路徑

    -path
    
open  > 打開檔案

    -open ex.txt
